# InMezzoApp (2018-2019)

A Social Network for a choir.

Functionalities:
* News Feed
* Music Portfolio
* Contacts
* About
* Concerts
* Members
* Appliances
* Reserves

Only For Memberes (Logged In)
* Profile
* Warnings 
* Rehearsals
* Chat
